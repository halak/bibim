EndOfJson

EndOfJson — Indicates whether the reader has reached the end of the input stream

Description

public bool EndOfJson { get; }

This property tells the user if the reader has reached the end of the input.

Note that reading data from a JsonReader can be done in a loop checking the return value of the JsonReader.Read() method, in which case it's not necessary to use this property.

Examples

Example 1.3. Checking the EndOfJson property

using LitJson;

class JsonSample
{
    void ReadJson (string json_data)
    {
        JsonReader reader = new JsonReader (json_data);

        while (! reader.EndOfJson) {
            reader.Read ();

            // Do something here
        }
    }
}
