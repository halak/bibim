A.2. Grammar

Since the lexer handles most of the hairy stuff (returning it as terminals), the resulting grammar deals mainly with the recursive portions.

A.2.1. LL(1) Form

TextS -> Object
       | Array

Object  -> { Object'
Object' -> }
         | Pair PairRest }

PairRest -> ε
          | , Pair PairRest
Pair     -> String : Value

Array     -> [ Array'
Array'    -> ]
           | Value ValueRest ]
ValueRest -> ε
           | , Value ValueRest

Value -> String
       | Object
       | Array
       | NUMBER
       | TRUE
       | FALSE
       | NULL

String -> " CHARSEQ "
