
Microsoft VisualStudio 제품군에서 Debugging할 때,
변수 조사식 창에 Bibim Engine의 class들의 Field가 보기 좋게 표시되게 해주는 Visualizer Codes.

사용법

1. 동봉 되어 있는 "VisualStudio Debugging Visualizers.txt"에 있는 내용을 복사합니다.
2. Notepad로 (VisualStudio Directory)\Common7\Packages\Debugger\autoexp.dat를 엽니다.
3. [Visualizer] Section 중 적당한 곳에 붙여넣기 합니다.
   (잘 모르겠으면 [Visualizer] 바로 밑줄에 붙여넣습니다)
